## 📌 Sobre o projeto

Este projeto foi desenvolvido durante o módulo de **Responsividade** da formação Fullstack da Rocketseat, com o objetivo de aplicar, na prática, conceitos modernos de construção de interfaces web.
A proposta consiste na criação de uma ***landing page para o aplicativo fictício de karaokê Zingen***, com foco no desenvolvimento de uma interface bem estruturada, escalável e adaptável a diferentes dispositivos.

Durante o desenvolvimento, foram explorados:<br>
🔹 Estruturação semântica com HTML<br>
🔹 Organização e escalabilidade do CSS<br>
🔹 Construção de layouts responsivos modernos<br>
🔹 Aplicação de boas práticas de desenvolvimento front-end

🔗 **Acesse o projeto online:**  
https://mdouracyfreire.github.io/zingen/

## 🧠 Principais conceitos aplicados

### Arquitetura CSS modular
O projeto utiliza uma estrutura modular baseada em múltiplos arquivos CSS:<br>
🔹 index.css (orquestra os imports)<br>
🔹 global.css (design system)<br>
🔹 utility.css (classes utilitárias)<br>
🔹 buttons.css (componentes)<br>
🔹 cards.css (componentes)<br>
🔹 social.css (componentes)<br>
🔹 header.css / hero.css / sections.css / etc (layout)

---

### Design System com CSS Variables
Foi implementado um sistema de design utilizando variáveis CSS:<br>
🔹 Consistência visual  
🔹 Facilidade de manutenção  
🔹 Escalabilidade  

---

### Utility-first CSS
Utilização de classes como:<br>
🔹 .flex<br>
🔹 .grid<br>
🔹 .gap-1<br>
🔹 .py-xl<br>
🔹 .container, etc

---

### Componentização com CSS
Componentes reutilizáveis como:<br>
🔹 Botões<br>
🔹 Cards<br>
🔹 Ícones sociais<br>

---

### Uso avançado de CSS
🔹 CSS Grid
🔹 Flexbox
🔹 Variáveis
🔹 Pseudo-elementos

---

### Responsividade
🔹 Mobile-first
🔹 Media queries modernas
🔹 Layout adaptável

## 🧰 Como Executar Localmente

**1. Clonar o repositório**
```bash
git clone https://github.com/mdouracyfreire/zingen.git
```
**2. Entrar na pasta do projeto**
```bash
cd zingen
```
**3. Abra o arquivo index.html diretamente no navegador ou utilize a extensão Live Server do VS Code**
